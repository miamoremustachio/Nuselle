# Useful commands

| _Command_                                        | _Usage_                                   |
| ------------------------------------------------ | ----------------------------------------- |
| `yt-dlp -U`                                      | Update                                    |
| `yt-dlp --version`                               | Show version                              |
| `yt-dlp -F`                                      | Show formats list                         |
| `yt-dlp -S ext`                                  | Download with format sorting              |
| `yt-dlp -S ext,proto`                            | Download with format & protocol sorting   |
| `yt-dlp -S ext,res:1080`                         | Download with format & resolution sorting |
| `yt-dlp --write-thumbnail --skip-download`       | Download thumbnail only                   |
| `yt-dlp -x`                                      | Extract audio and download it             |
| `yt-dlp -x --add-metadata --embed-thumbnail`     | Download with metadata/cover added        |
| `yt-dlp -x --audio-format mp3 --audio-quality 0` | Download as mp3 with the best quality VBR |
| `yt-dlp --merge-output-format mkv`               | Merge download into MKV container         |
