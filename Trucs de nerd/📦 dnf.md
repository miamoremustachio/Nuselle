## Basic commands

| _**Command**_                | _**Description**_                                                 |
| ---------------------------- | ----------------------------------------------------------------- |
| `dnf upgrade`                | upgrade all packages                                              |
| `dnf install` \| `in`        | install the package                                               |
| `dnf reinstall` \| `rei`     | reinstall the package                                             |
| `dnf remove` \| `rm`         | remove the package                                                |
| `dnf autoremove`             | removes all packages from the system which are no longer required |
| `dnf downgrade` \| `dg`      | downgrades the specified packages                                 |
| `dnf search`                 | search the package in the repository                              |
| `dnf info` \| `if`           | get info about the package                                        |
| `dnf list` \| `ls`           | display the list of the packages                                  |
| `dnf repolist`               | show all the available repositories                               |
| `dnf deplist`                | show the dependencies of the package                              |
| `dnf history` \| `hist`      | show the history of the packages                                  |
| `dnf alias add <name=value>` | create new alias                                                  |
| `dnf alias delete <name>`    | delete alias                                                      |
| `dnf alias list`             | list all defined aliases                                          |
